# RSSHub Home Assistant Addon

Generate RSS feeds for everything with RSSHub integrated into Home Assistant.

## Features

- 📡 Generate RSS feeds from any website
- 🔒 HTTPS support through Home Assistant ingress
- 🎯 Sidebar integration via ingress mode
- 🌐 Host network access for full connectivity

## Installation

1. Add this repository to your Home Assistant addon store
2. Install the RSSHub addon
3. Start the addon
4. Access RSSHub through the Home Assistant sidebar or at `http://homeassistant.local:1200`

## Configuration

The addon runs on port 1200 by default. No additional configuration is required.

## Usage

Once installed, you can:
- Access RSSHub from the Home Assistant sidebar (look for the RSS icon)
- Use RSSHub API at `http://homeassistant.local:1200`
- Generate RSS feeds for any supported source

## Support

For RSSHub documentation and supported routes, visit: https://docs.rsshub.app/
